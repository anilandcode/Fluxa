# Architecture: Fluxa

## Apps
- **demo-site**: Showcase of Fluxa capabilities.
- **webflow-component**: Bridge for importing React components into Webflow.

## Packages
- **@fluxa/motion-engine**: Animation utilities and wrappers.
- **@fluxa/ai-engine**: AI-powered logic and agents.
- **@fluxa/layout-engine**: Advanced layout algorithms.
- **@fluxa/ui-system**: Core design system and primitive components.
- **@fluxa/presets**: Pre-configured design tokens and themes.
